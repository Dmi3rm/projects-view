Logic Scheme Builder

Приложение позволяет строить и исполнять логические цепочки.

Поддерживаются 3 логических элемента:

  • - ИЛИ (OR)
  • - И (AND)
  • - НЕ (NOT)

Для добавления элемента в схему следует нажать на его изображение на панели управления, после чего кликнуть на поле - элемент появится под курсором.

Например, добавим на поле элементы НЕ и ИЛИ. Чтобы приступить к настройке схемы, в меню приложения нажимаем Mode -> Setting.

Для перемещения элемента по полю, зажмите курсор в его центре и потяните в нужное место. Переместите элемент в правый верхний угол для его удаления.
Кликните по входу элемента, чтобы изменить его начальное значение.
Зажмите курсор на выходе элемента, потяните на вход другого и отпустите для создания логической связи. Кликните по линии связи чтобы ее удалить.

Для управления выполнением схемы на панели управления есть 3 кнопки:
- выполнить схему до конца;
- выполнить только следующий шаг;
- откатить схему к начальным значениям.
